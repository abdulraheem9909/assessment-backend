## Description

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Some of packages used in this project

1. For Send Email
   @nestjs-modules/mailer
   handlebars (To send custom template with email when user signup)

2. For authentication
   passport
   passport-jwt
   passport-local

3. For valication
   class-validator

4. Encryption
   bcrypt
